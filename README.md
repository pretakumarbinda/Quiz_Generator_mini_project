# Note Summary & Quiz Generator

An AI-powered web application that converts handwritten note images into concise summaries, audio explanations, and quiz questions.

## Features

- Upload up to 3 handwritten note images
- Generate AI-based note summaries
- Convert summaries into audio
- Generate quizzes with different difficulty levels
  - Easy
  - Medium
  - Hard
- User-friendly interface with Streamlit
- Preview uploaded images before processing

- Python
- Streamlit
- PIL (Python Imaging Library)
- Generative AI API
- Text-to-Speech API

## Project Workflow

1. Upload note images
2. Select quiz difficulty level
3. Generate AI note summary
4. Generate audio summary
5. Generate quiz questions
